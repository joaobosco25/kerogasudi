# Kero Gás UDI - Site estático

Site HTML/CSS/JS inspirado no mockup gerado para uma revendedora de gás parceira Supergasbras.

## Estrutura

- `index.html`: página principal.
- `assets/css/style.css`: todo o visual do site.
- `assets/js/main.js`: menu mobile e rolagem suave.
- `assets/img/hero-visual.png`: composição visual usada no hero.
- `assets/img/mockup-reference.png`: imagem de referência do modelo aprovado.

## Como abrir

Abra o arquivo `index.html` diretamente no navegador ou rode com Live Server no VSCode.
